# Support FAQ — Payments

## Payout Delays
**Q:** When do we escalate?  
**A:** If delay > 3 hours or backlog age > 25 minutes. See SLA.

## Limits
**Q:** Can a Gold user increase daily P2P limit?  
**A:** Yes; they’re eligible up to $5,000/day. If profile unchanged, no repeat KYC.

## API
**Q:** Where’s the batch payouts example?  
**A:** See “Payouts API v2” and the OpenAPI JSON for request/response formats.

## Refunds
**Q:** Can we refund after a chargeback?  
**A:** No—block to prevent double credit. Consult Refunds Policy.
